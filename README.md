# TryHackMe-CheeseCTF-Walkthrough
End-to-end penetration testing walkthrough demonstrating network enumeration, web exploitation, Linux privilege escalation, and root compromise on a TryHackMe CTF machine.


Project Overview

CheeseCTF is a hands-on penetration testing project performed on a vulnerable Linux machine hosted on TryHackMe.
The project follows an end-to-end ethical hacking methodology, covering reconnaissance, enumeration, exploitation, privilege escalation, and post-exploitation.

This repository demonstrates practical cybersecurity skills aligned with SOC Analyst / Penetration Tester / Security Analyst roles.
